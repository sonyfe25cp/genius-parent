北京理工大学法学院09310701班喜获北京市学习型“十佳示范班级”荣誉称号
　　6月20日，由北京市教工委主办的北京市首届我的班级我的家学习型班集体创建示范活动在华北电力大学（北京）隆重举行。经校学生工作处推选，法学院09310701班作为优秀班级代表参加了此次评选。在评选过程中凭借出色的表现征服了在场的评委和观众，将学习型十佳示范班级的荣誉称号收入囊中。参加本次评选的还有北京大学、清华大学等五十五所首都知名院校的优秀班级代表。
　　此次北京市十佳示范班级评比共包含三个环节：班级风采展区、舞台风采展示以及班级建设答辩。活动现场，09310701班的同学们分工有序、团结协作，以饱满的热情和昂扬的风貌向众人展示着北理学子的风采。
　　在班级风采展区处，同学们将印有全体同学合影及班级党团、学习、生活等各类活动简介的喷绘和展板布置在展区四周，丰富的内容吸引了众多评选老师以及其他院校学生的参观和欣赏。同时，班级特别安排专门的解说人员，为前来参观的师生讲解展板内容。
　　在舞台风采展示环节，09310701班的表演给评委留下深刻印象。09310701班舞台展示的主题为相亲相爱一家人，有两大亮点：一是立意深刻，不但传达了班级内部同学之间互帮互助、相亲相爱的感情，更凸显了该班在西南旱情严重时期积极承担社会责任、奉献大爱的担当精神；二是形式新颖，打破传统的合唱、跳舞或单一朗诵的模式，大胆地将情景表演与朗诵相结合，并在最后借助班歌将整个表演推向高潮。
　　在答辩环节，09310701团支部书记姚书佳同学作为答辩人，就本班的班训、党团文化建设、学习文化建设等内容上台进行答辩。深刻的内容、流利的讲说、敏捷的思维以及全班同学在答辩现场所表现出的团结，均是此次评选活动取得优异成绩的重要因素。
　　法学院09310701班全体同学入学三年来，在校院领导、老师的关怀与指导下不断成长、不断进步，以一系列市、校级荣誉为依托，竞争本次评选，最终摘得学习型十佳示范班级的荣誉称号。这不仅是对一个班级的肯定，更是对北京理工大学以及北京理工大学法学院施教育人成果的肯定。希望09310701班能够继续秉承我校团结、勤奋、求实、创新的校训为我校乃至北京市的院校班级起到良好的学习示范作用。
　　1.北京理工大学法学院09310701班所获荣誉
　　　　2009年北京理工大学优良学风宿舍
　　　　2009年北京理工大学优秀团支部
　　　　2009年北京理工大学优秀班集体
　　　　2010年北京理工大学十佳团支部
　　　　2010年北京市先锋杯优秀团支部
　　2.北京理工大学法学院09310701班训修身致用，以法经国。
　　修身，欲明明德于天下者，先须修身齐家。修身是将大学生自身成长成才放在了班级建设的首位。所谓修身，不仅包括科学文化知识的学习，更包含道德层面的教育。
　　致用，大学教育应当以培育能够将知识应用于现代化建设的人才为目的，大学生应当尽可能培养自己适应社会、学以致用的能力。
　　以法，我班是一个法学专业班级，班旨将以法纳入，一是体现专业特色，提醒班级成员时刻不忘自己是一名法律人；二是将法律中公平、正义的理想和原则带入班级文化建设。
　　经国，经国济世应当是知识分子的最高追求，也是当代大学生应有的时代使命感。
