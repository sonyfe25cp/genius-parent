第十九次全国高等学校党的建设工作会议闭幕
《中国教育报》2010年12月16日第1版
本报北京12月15日讯（记者 唐景莉 杨晨光）第十九次全国高等学校党的建设工作会议今天在京闭幕。教育部党组书记、部长袁贵仁在会议总结讲话中强调，要深入贯彻落实党的十七届五中全会精神、全国教育工作会议精神、全国人才工作会议精神和教育规划纲要、人才规划纲要，贯彻落实新颁布的《中国共产党普通高等学校基层党组织工作条例》，按照习近平同志和刘延东、李源潮同志重要讲话要求，整体推进，突出重点，努力开创高校党建工作新局面。
袁贵仁指出，会议期间，习近平同志和刘延东、李源潮同志亲切会见了与会代表并合影留念，习近平同志即席发表了重要讲话，刘延东同志出席会议并作重要讲话，李源潮同志主持会议并作出重要部署。这充分体现了党中央、国务院对高校党建工作的高度重视、亲切关怀，对高等教育事业的殷切期望。中央领导同志的重要讲话，站在党和国家事业发展全局的高度，精辟阐述了加强高校党的建设的重大意义，明确了围绕中心抓党建的工作思路，提出了贯彻落实新颁布的《条例》的任务要求，对切实提高高校党建工作的科学化水平，全面推动高等教育事业科学发展，具有十分重要的意义。各地各高校要深刻领会这次会议的主要精神，把思想认识统一到中央对高校党建工作形势的科学判断上来，统一到促进高等学校科学发展、和谐稳定上来，统一到中央对高校党建工作的决策部署上来。
袁贵仁强调，贯彻落实会议精神，要认真贯彻十二五规划建议和教育规划纲要对高等教育提出的新要求，把提高质量这一核心任务落实到位；以立德树人为根本，把加强和改进大学生思想政治教育工作的各项任务落实到位；着眼于为高等教育事业科学发展提供动力，把深入开展创先争优活动落实到位；切实抓好《条例》的学习贯彻，把加强基层党组织建设的各项任务落实到位；不断增强政治意识、大局意识、责任意识，把促进校园和谐稳定的各项任务落实到位。袁贵仁要求，各地各高校要及时汇报传达，抓紧研究部署，强化督促检查，树立良好作风，扎扎实实做好落实会议精神的各项任务，以优异成绩迎接建党90周年。
闭幕会议由中央宣传部副部长申维辰主持。中央组织部副部长王秦丰，中央组织部部务委员兼组织二局局长陈向群，教育部党组成员、部长助理林蕙青出席会议。中共北京市委教育工委、中共江苏省委教育工委、中共上海市民办高校工委、上海交通大学、华中农业大学、湖南大学、北京理工大学、辽宁大学、郑州大学、塔里木大学、宁波职业技术学院等11个单位的代表在会上作了交流发言。
12月14日至15日，第十九次全国高等学校党的建设工作会议在京召开。把握和用好重大战略机遇期，推动科学发展和加快经济发展方式转变，对高校党的建设提出了新使命；推动高等教育科学发展，加快建设中国特色现代高等教育体系，对高校党的建设提出了新任务；应对意识形态和思想文化领域的新挑战，维护高校和谐稳定的大好局面，对高校党的建设提出了新课题。与会代表认为，面对新使命、新任务、新课题，要把思想和行动统一到中央的决策部署上来，把高校党建提高到一个新的水平。
　　一份份饱含深情的入党申请书立下滚烫誓言，一面面鲜红的党旗引领青春征程。近年来，华中科技大学每年都有90％以上的新生递交入党申请书，这与学校实施的党旗领航工程是分不开的。通过早日站在党旗下、党旗在我心中、我为党旗添光彩等活动，青年学生的入党热情被极大地激发出来。截至2009年12月，学校共有本科生、研究生党员1.6万多人，占全校学生总数的31.5%。
　　从北京奥运会到新中国成立60周年，从上海世博会到广州亚运会，从抗震救灾到应对国际金融危机，无论是大事喜事，还是急事难事，高校党组织和党员都发挥了突出作用，展示出良好的思想政治素质和精神风貌。
　　高校党建是党的建设新的伟大工程重要组成部分，在整个党的建设中具有特殊重要的地位。从1990年开始，中央组织部、中央宣传部和教育部党组基本每年联合召开一次全国高校党的建设工作会议，研究部署高校党建工作重大问题，到今年已经是第十九次。
　　在中央的正确领导下，各级党委和高校党组织加强和改进党的建设，自觉围绕中心，服务大局，发挥了学校党委的领导核心作用、基层党组织的政治核心作用、战斗堡垒作用和党员的先锋模范作用，保证了高校正确的办学方向，推进了高校的改革发展，维护了高校连续20多年和谐稳定的局面。
　　中国农业大学通过一个博士生党支部与一个村党支部结对共建、一名博士生党员帮扶一个农户，开展红色１＋１科技行动，在广袤的农村大地上，博士生们送去了知识和技术，收获了素质的提升和责任意识的增强。在他们的实践活动中，党建与育人得到了有机的统一。
　　北京大学学生党建工作形成了一套完善的、行之有效的体系和规范，分别在入党积极分子的培养阶段、在党员发展工作中以及在进入党组织后开展特色鲜明、吸引力强、激励作用明显的主题教育活动。这些分阶段、有针对性的教育活动和课堂教学组织工作一样，具有规范性、科学性，是新时期加强和改进高校育人工作的有效途径。
　　教育部组织的2009年高校学生思想政治状况滚动调查显示，教育部直属高校在校大学生党员占在校生总数的比例从1990年的1.2%提高到2008年的11%，19年增加了近10个百分点。据统计，2009年，全国高校在大学生中发展党员118万人，占全国党员发展总数的38．5%，源源不断地为党的队伍注入新的血液。
高校党的建设不断推进，为高等教育事业实现持续健康发展提供了坚强保障。中国建成了在学人数3000万人、世界上最大规模的高等教育体系，十一五期间，高等教育毛入学率从21%提高到26%，为国家输送了3429万名毕业生，其中有145万名硕士生和近22万名博士生，超过过去20年的总和，教育质量、科研水平与社会服务能力稳步提升。
　　今年，党的十七届五中全会召开，作出了当前和今后一个时期我国发展仍处于可以大有作为的重要战略机遇期的科学判断。新世纪第一次全国教育工作会议的召开和教育规划纲要的颁布，对今后10年教育改革发展作了总体谋划，特别提出把加强组织领导作为重要保障措施。这几件关系现代化建设全局的大事，对高等教育的改革发展有着直接而深远的影响，也对高校党的建设提出了新的要求。
　　围绕中心抓党建，抓好党建促发展。新形势下，学校党委把加强党的建设作为推进学校发展的原动力，认真贯彻全国教育工作会议精神和教育规划纲要，围绕制定学校未来10年发展规划，积极开展高水平大学建设大讨论。郑州大学党委书记郑永扣说，我们要力争成为国家特别是河南经济社会发展的重要支撑力量。
　　学习组织体系化、学习内容精细化、学习平台立体化、学习方式多样化、学习考评规范化这是湖南大学深入推进学习型党组织建设而采取的主要做法。近一年来，学校紧紧围绕学习贯彻中央关于教育工作的重大决策部署和教育改革发展最新形势，在学校中层干部、基层党支部书记、辅导员班主任和学生代表等各层面，举办各类专题学习110多场次。该校党委书记刘克利说：我们把学习作为学校各级党组织和党员干部的政治责任、精神追求和工作要求来落实，作为贯彻落实党的十七届五中全会和全国教育工作会议精神的重要抓手来推进，取得初步成效。今年，学校先后荣获湖南省书香校园和湖南省建设学习型党组织示范点称号。
　　各地各高校立足高等教育改革发展的新任务新要求，瞄准大目标，找准切入点，把握着力点，使高校党的建设与高等教育事业科学发展紧密结合、协调推进。
　　中共北京市委教育工委根据事业发展需要，优化党组织设置，主动适应学校实体单位新变化，在学科组、创新团队、重点实验室等领域建立党组织，及时推广教师乐业工程、学生党员德育答辩等一大批好做法，确保党组织有机融入中心工作、有效发挥作用。
　　学校党组织与学校同步建立、党组织负责人与校长同步落实、党建工作与行政工作同步安排，这是中共上海市民办高校工委在推进民办高校党组织建设中采取的措施。目前，上海民办高校党员有4960多名，其中教职工党员在教职工中占38.2%，学生党员占3.4%，经各级党校培训的入党积极分子近万名，提出入党申请的学生将近半数。
　　党风正，人心聚。党建工作与学校改革发展互动推进的良好局面正在高校形成。我校已有12个学科进入美国科技信息所《基本科学指标（ESI）》前1%。上海交通大学党委书记马德秀说，坚持党委统一领导，党政团结合作，保证了事业发展的正确方向，使学校呈现出积极进取、奋发向上的气象。
　　高校基层党组织是党在高校全部工作和战斗力的基础。今年8月，中央正式印发《中国共产党普通高等学校基层组织工作条例》，贯彻了中央对高校党的工作的新要求，成为高校推进党建工作的基本规章。
　　辽宁大学党委书记王山说：认真学习贯彻落实《条例》，对提高高校基层党组织建设的科学化水平具有重要意义。
　　按照《条例》要求，中共江苏省委教育工委完善体制机制，打造坚强党委领导核心，将高校党委抓党建工作情况、贯彻党委领导下的校长负责制情况作为考核重点，集中对40所本科高校和37所高职高专进行了考核，实现了从重在规范化建设向重在提升科学化水平的转变。
　　努力把基层党组织建设提高到一个新水平，各高校采取了很多举措。
　　在实施一院带一村、辐射一个县服务新农村建设111行动计划中，华中农业大学组织以39位国家现代农业产业体系岗位科学家为核心的一批党员教师，深入农村最基层、生产最前沿，60多个农业科技团队常年工作在田间地头，与地方村党支部结对建立22个联合党支部，形成了党建与新农村建设紧密结合、互促共进的新机制。除了校地联合型党支部，学校还在28个科研基地，组建了近20个学术联合型党支部。该校党委书记李忠云说，这些都是学校创新基层组织建设、服务国家战略需求的尝试，通过党组织的凝聚力来增强团队的创新能力，科技攻关能力明显提升。据了解，十一五以来，该校承担国家级项目900多项，与华中地区70个县（市、区）建立了农科教合作关系，以杂交油菜、优质柑橘等为代表的一批重大科研成果，为我国农业科技创新和经济社会发展作出突出贡献。
　　塔里木大学党委书记王选东说：学校把强化反分裂、反渗透意识作为领导班子建设的政治任务，把民族团结教育作为加强师德的主要内容，与课堂教学评价和各专业教学相结合，严格要求学术研究无禁区、课堂讲授有纪律。
　　在北京理工大学开展的党群零距离活动中，各级党组织通过广结对子、谈心交心、比学赶超等活动，促进党员、党群之间多角度、全方位的沟通交流，为培养人才搭建平台。该校党委书记郭大成说：党群零距离工程覆盖了学校各级党组织，延伸到学校工作的方方面面，已成为学校创先争优和密切党群关系的常态活动。
　　宁波职业技术学院以开展创先争优活动为契机，建立三个一联系制，每个党员教师主动联系一个班级、一个寝室、一名家庭经济困难学生，对学生的思想、学习、就业创业等提供指导与帮助，用实际行动助推学生成长成才。
　　以创先争优活动为抓手，在日常工作中彰显先进，在党组织的带动引领下，广大师生学习先进、争当先进的良好风气正在校园里形成。高校党建工作焕发出新活力。
