机械与车辆学院2010年研究生开学典礼隆重举行_北京理工大学新闻网
 
　　9月17日，机械与车辆学院研究生开学典礼在七号楼报告厅隆重召开，机械与车辆学院院长项昌乐教授、分党委书记张英教授、副院长左正兴教授、胡纪滨教授出席了本次会议。会议由副书记、副院长阎艳教授主持。
会议在庄严的国歌声中拉开帷幕
分党委书记张英教授致辞
　　张英书记对所有的新同学表示欢迎，并祝贺大家从众多的考生中胜出进入到了北京理工大学机械与车辆学院。
院长项昌乐教授介绍学院基本情况
　　项昌乐教授对学院的基本情况，包括科研、师资力量等方面进行了简单的介绍，从宏观上就研究生阶段如何进行学习科研等方面为新生做了讲话，希望大家能珍惜研究生这一学习阶段，为学院发展，为祖国建设做出自己的贡献。
　　开学典礼结束后主管研究生的副院长左正兴教授与同学们进行了交流，并对同学们提出了希望。
（审核：阎艳）
